## What Improved

CI now covers the deterministic product path beyond the original build/test/validate gates: evaluation, doctor diagnostics, prompt routing, goal planning, scope contract creation, assessment-pack generation, and self-loop assessment-pack-only evidence creation.

The workflow no longer runs the test suite twice. `npm run coverage` already invokes `scripts/run-tests.mjs`, so CI uses that single coverage-backed test pass and documents the intent in the workflow and CI docs.

## What Did Not Improve

CI still does not run mission execution, Codex, provider adapters, PR creation, or full goal execution. That is intentional for safety and determinism. The smoke commands create local generated artifacts in the CI checkout, but the workflow does not validate their diff as durable committed memory.

## Risks

The goal and scope smoke commands write local artifacts under `.abstraction-tree/` during CI. They are acceptable as scratch output, but future CI additions should avoid adding a final broad `git diff --exit-code` unless those generated paths are excluded or redirected.

## Missing Context Discovered

`npm run coverage` is a wrapper around `scripts/run-tests.mjs`; keeping `npm test` immediately after it was redundant.

PowerShell still blocks `npm.ps1` on this machine, so local verification used `npm.cmd`.

## Tree Updates Needed

Completed. `npm.cmd run atree:scan`, `npm.cmd run atree:validate`, and `npm.cmd run atree:evaluate` refreshed the repository memory after CI and docs changes.

## Reusable Lesson

CI should smoke test the product workflow, not only the library internals. When a repo's main value shifts toward routing, planning, scope control, and review artifacts, deterministic smoke commands should protect those paths without invoking agents or external services.

## Recommended Next Loop

Add a small script-level CI smoke helper that runs the route/goal/scope/assessment commands with temporary output paths, making the GitHub Actions YAML shorter and reducing the chance that future generated artifacts accidentally become part of a diff gate.
