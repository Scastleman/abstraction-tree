# Lesson: Autonomy Claims Belong In Validation

Claims about autonomy, self-improvement, or autonomous correctness are safety-sensitive product language. A small lint rule can catch unqualified public claims before they reach README, docs, package READMEs, adapter docs, or current agent prompts.

Allow explicit non-goals and historical terminology, but keep archived run reports and lessons out of scope so old operational memory is not rewritten just to satisfy current phrasing.
